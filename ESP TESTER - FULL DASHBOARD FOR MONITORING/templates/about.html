üéØ Project Overview

The Smart Campus Security Model is an advanced IoT cybersecurity research platform that simulates a miniature connected campus environment. This project demonstrates real-world offensive and defensive cybersecurity techniques applied to Internet of Things (IoT) infrastructure.

Our system integrates multiple ESP32 microcontroller nodes deployed across four distinct campus zones, each equipped with specialized sensors for environmental monitoring, security, and automation. The entire network is controlled by a Raspberry Pi 4 acting as the central server, database, and security monitoring hub.

üõ°Ô∏è Cybersecurity Research

Offensive Security (Red Team Operations)

This project simulates real-world cyberattacks on IoT infrastructure to identify vulnerabilities and test defensive measures. Attack scenarios include:

  • Denial of Service (DoS): Flooding ESP32 nodes with excessive HTTP requests to overwhelm resources
  • SQL Injection: Testing database input validation and sanitization
  • Brute Force Attacks: Credential stuffing using tools like Hydra and Crunch
  • Man-in-the-Middle (MITM): ARP spoofing to intercept network communications
  • Network Reconnaissance: Using Nmap for vulnerability scanning

Defensive Security (Blue Team Operations)

Comprehensive monitoring and mitigation strategies implemented:

  • Real-time Packet Analysis: Monitoring incoming/outgoing network traffic
  • Anomaly Detection: Identifying unusual patterns in sensor data and network behavior
  • Firewall Rules: UFW configuration for access control and port filtering
  • Fail2Ban Integration: Automatic IP blocking after repeated failed login attempts
  • Database Hardening: Prepared statements and input validation
  • TLS Encryption: Secure HTTPS communication for dashboard access
  • System Health Monitoring: CPU usage, memory, and network load tracking
